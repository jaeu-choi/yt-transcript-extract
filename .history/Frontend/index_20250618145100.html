✨ 사용법

  1. YouTube 동영상 페이지에서 URL을 복사합니다
  2. 위의 입력란에 URL을 붙여넣습니다
  3. '언어 확인' 버튼을 클릭하여 사용 가능한 언어를 확인합니다
  4. 원하는 언어를 선택합니다
  5. 타임스탬프 포함 여부를 선택합니다
  6. '자막 추출' 버튼을 클릭합니다
  7. 자막이 추출되면 클립보드로 복사할 수 있습니다

🎯 지원 형식

  • 모든 언어 자막 (한국어, 영어 우선 표시)
  • 자동 생성 자막 포함
  • 타임스탬프 포함/제외 선택 가능
  • 다양한 YouTube URL 형식 지원
  • 언어별 자막 선택 가능
